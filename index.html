suriyaaOS

A simple, basic, open-sourced UNIX-like operating system by Suriyaa Kudo.

  • Install suriyaaOS

Specifications & Features

The OS was built for the x86 architecture, running on 32 bits, and compatible with IBM PCs. It also works as a virtual machine nearly on all computers like on Windows, Mac & Linux!

  • Read The Installation Documentation
  • Coded in C++
  • x86-architecture (32-bits)
  • Boot with (GNU) Grub
  • Kind of modular system for drivers
  • Kind of UNIX style
  • Multitasking
  • ELF executable in userland
  • Modules
  • Userland
  • "Can" run a shell or some excutables like Lua
